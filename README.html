WPF.QuickStart Caliburn - 2014

Designed by @Daniel NGUYEN :smirk_cat:

WPF quickstart caliburn is a ready-to-use project that uses :

  • WCF Services for Client/Server communication in duplex
  • MahApps.Metro for UI design
  • Caliburn.Micro to setup MVVM
  • Twitter API
  • Yahoo! Finance API

Installation Notes

Server side Console:

Launch the server side project executable.

Client Side UI:

Launch the client. :heavyexclamationmark: In order to be allowed connection and retrieved data, the server console must be started.
